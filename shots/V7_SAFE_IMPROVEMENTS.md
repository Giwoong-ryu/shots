## ✅ 안전하게 적용 가능한 개선 사항

### 1. 자막 위치 최적화 ⭐⭐⭐⭐⭐

**현재 상태**: 
- `쇼츠_고급.json`에서 자막 위치 확인 필요

**개선 사항**:
```javascript
// FFmpeg ASS 자막 생성 시
// Alignment 값 변경
// 2 = 하단 중앙 (현재)
// 8 = 상단 중앙 (권장)

Style: Default,Arial,48,&H00FFFFFF,&H000000FF,&H00000000,&H00000000,
-1,0,0,0,100,100,0,0,1,3,3,8,10,10,10,1
                           ↑
                        여기를 8로
```

**이유**:
- 모바일에서 하단 UI 요소(좋아요, 댓글 등)에 가려지지 않음
- 시청자 집중도 향상
- **기존 로직 변경 없음** (위치만 조정)

**적용 방법**:
- `Generate ASS Subtitles` 노드의 `Alignment` 값만 수정
- 다른 로직은 **그대로 유지**

---

### 2. BGM 볼륨 최적화 ⭐⭐⭐⭐

**현재 상태**:
- `쇼츠_고급.json`에서 BGM 볼륨 30% (0.3)

**개선 사항**:
```javascript
// 현재: volume=0.3 (30%)
// 권장: volume=0.25 (25%)

// 이유: 목소리가 더 명확하게 들림
// 블루투스 이어폰 기준 최적 비율
```

**적용 방법**:
- `Music random1` 노드의 `volume` 값만 0.3 → 0.25로 수정
- **기존 FFmpeg 로직은 그대로**

---

### 3. 크리에이티브 커먼즈(CC) 백업 소스 ⭐⭐⭐

**현재 상태**:
- NanoBanana/Sora-2로 이미지/비디오 생성

**개선 사항**:
- **백업 전략**으로만 활용
- API 실패 시 대체 소스로 사용
- **기존 AI 생성 로직은 유지**

**적용 방법**:
```javascript
// 새로운 노드 추가 (선택사항)
// "Fallback: CC Video Search"
// NanoBanana/Sora-2 실패 시에만 실행
```

**주의**:
- **메인 로직 변경 없음**
- 단순 백업 용도
- 비용 절감 효과

---

### 4. 자막 테두리 강화 ⭐⭐⭐

**현재 상태**:
- 테두리 두께 확인 필요

**개선 사항**:
```javascript
// ASS 자막 스타일
// Outline (테두리): 2 → 3
// Shadow (그림자): 2 → 3

Style: Default,Arial,48,&H00FFFFFF,&H000000FF,&H00000000,&H00000000,
-1,0,0,0,100,100,0,0,1,3,3,8,10,10,10,1
                         ↑ ↑
                    테두리 그림자
```

**이유**:
- 가독성 향상
- 전문성 증가
- 배경과 명확한 구분

---

### 5. 영상 길이 유지 (변경 없음) ✅

**현재 상태**:
- 6개 장면 × 5초 = 30초 (이미 최적!)

**판단**:
- **현재 길이 완벽함**
- 40초 이내 권장 기준 충족
- **변경 불필요**
