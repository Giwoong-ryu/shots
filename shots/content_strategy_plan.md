# 📈 고수익 시니어 쇼츠 전략 (프로젝트 보람이)

## 1. 핵심 컨셉 및 페르소나 전략
**목표**: 시니어들이 **신뢰하고**, **끝까지 시청하며**, 친구들에게 **공유하는** 콘텐츠 만들기.

### 🎭 페르소나: "보람이"
*   **정체성**: 5살 손주 (똑똑하고, 귀엽고, 예의 바름).
*   **성공 요인**:
    *   **신뢰**: 어르신들은 아이들에게 경계심을 갖지 않습니다.
    *   **감성**: 손주에 대한 사랑과 그리움을 자극합니다.
    *   **권위**: "의사 선생님이 그랬는데요~" (직접 가르치려 들지 않고 간접적으로 전달).
*   **목소리 톤**: 아이 목소리지만 **느린 템포**와 **또박또박한 발음** 필수 (난청 고려).

## 2. 타겟 분석 (60대 이상 한국 시니어)
| 특징 | 상태 | 대응 전략 |
| :--- | :--- | :--- |
| **시력** | 노안, 초점 맞추기 힘듦 | **초대형 자막**, 고대비(검은 배경에 노란/흰 글씨), 느린 화면 전환 (3-5초). |
| **청력** | 고주파 난청 | **중저음 또는 또박또박한 아이 목소리**, 배경음악은 작게. |
| **주의력** | 인내심은 있으나 복잡한 건 싫어함 | **영상 하나에 메시지는 딱 하나만**. 복잡한 논리 비약 금지. |
| **관심사** | 건강, 장수, 가족, 트로트 | 모든 건강 팁을 "오래 살아서 손주 봐야죠"와 연결. |

## 3. 수익화 공식 (시청 지속 시간 & 클릭률)
수익을 내려면 **평균 시청 지속 시간(AVD) 80% 이상**이 필요합니다.

### 🎣 훅 (Hook, 0-3초)
*   **규칙**: 인사는 생략하고 바로 **증상**을 건드려야 합니다.
*   **나쁜 예**: "안녕하세요, 오늘은 고혈압에 대해 알아보겠습니다." (이탈)
*   **좋은 예**: "할머니! 아침에 일어날 때 머리 핑~ 돌죠?" (시청)
*   **시각 효과**: 분할 화면 (아파하는 노인 이미지 + 놀란 보람이 얼굴).

### 🎬 몰입 구조 (30초 구성)
1.  **훅 (0-5초)**: 증상 + 공감 ("많이 아프셨죠?").
2.  **해결책 (5-20초)**: "비법" 공개 (누구나 할 수 있는 쉬운 행동, 예: "여기 꾹 누르세요").
3.  **약속 (20-25초)**: 효과 보장 ("그럼 이제 지팡이 없이 걸을 수 있어요!").
4.  **구독 유도 (25-30초)**: "보람이 또 보고 싶으면 구독 눌러주세요~" (귀여운 애교).

## 4. 워크플로우 구현 계획 (v2)

### 🧠 에이전트 1: "어그로" 기획자
*   **역할**: 검색량이 많은 주제 선정 (예: "당뇨에 좋은 음식", "무릎 관절").
*   **출력**: 제목, 훅 멘트, 썸네일 텍스트.

### ✍️ 에이전트 2: "몰입" 대본 작가
*   **역할**: [훅 -> 해결책 -> 약속] 구조에 맞춰 대본 작성.
*   **제약**: 문장은 짧게. 어려운 한자어 금지.

### 🎨 에이전트 3: 비주얼 감독
*   **역할**: 이미지 프롬프트 생성.
*   **규칙**:
    *   **캐릭터 일관성**: 보람이는 항상 같은 모습이어야 함 (의사 가운 입은 5살 여아).
    *   **시각적 비유**: "혈관이 수도관처럼 깨끗해져요" (직관적인 이미지).

### 🛠️ 기술 설정 (N8N)
*   **음성 (TTS)**: OpenAI `tts-1-hd` (목소리: `Nova` 또는 `Shimmer`, 속도: `0.85` - 천천히).
*   **이미지**: OpenAI DALL-E 3 (품질: `Standard` 가성비, 스타일: `Vivid` 선명하게).
*   **편집**: Creatomate (미리 **초대형 자막**이 설정된 템플릿 사용).
